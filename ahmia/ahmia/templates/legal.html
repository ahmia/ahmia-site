{% blocktrans %} This site is fully compliant with the Law of Finland. The administrator is removing any links to child abuse content. Please 1. email us the exact URL of the content you suspect to be child abuse, and 2. report anonymously through Save the Children Finland. {% endblocktrans %}

If a site user tries to search child abuse material the Ahmia.fi is not showing search results normally. Instead, the search offers a link to Self-help Program. The self-help program is primarily intended for people who are worried about their interest, thoughts, feelings or actions concerning children. If the search contains terms which are only used for searching abusive material then Ahmia.fi refuses to show the results. For example, „jailbait“ is one of these filtered terms. As a result, Ahmia.fi is offering a link to self-help program: https://ahmia.fi/search/?q=jailbait In addition, Ahmia applies automated text mining algorithm to remove possible links to child abuse from the search results.

Filtering of all sexual searchers

There is now a filtering of all sexual related searchers. This is because there are so many abusive websites. Using text-based classifiers, it is impossible to automatically filter out all child abuse websites. As a result, all sexually explicit searches are filtered.
